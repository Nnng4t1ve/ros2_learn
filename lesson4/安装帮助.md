在我的22.04环境下
libglut3.12和freeglut3有依赖问题
使用下面来解决

```
sudo apt install -y aptitude
sudo aptitude install ros-humble-turtlebot3-gazebo
```

或

```
The following actions will resolve these dependencies:

     Remove the following packages:                       

1)     libglut3.12 [3.4.0-4~22.04.sav0 (jammy, now)]      

     Install the following packages:                      

2)     freeglut3-dev [2.8.1-6 (jammy)]                    

     Keep the following packages at their current version:

3)     libglut-dev [Not Installed]  
```

启动新的Gazebo会加载比较久，所以不要动，动就会卡死。
等待画面弹出即可
